# ResQ — Emergency Response Assistant

AI-powered first aid guidance for East Africa, available in **English**, **Français**, and **Kiswahili**.

## Features
- Instant step-by-step first aid instructions via ChatGPT-4
- 8 quick-select emergency scenarios (cardiac arrest, choking, bleeding, burns, fracture, seizure, snake bite, drowning)
- Nearby hospital search using OpenStreetMap / Overpass API
- Emergency contact numbers for Rwanda, Kenya, Tanzania, and Uganda
- Light/dark theme with localStorage persistence
- Fully responsive for mobile use in the field

## Countries Covered
🇷🇼 Rwanda · 🇰🇪 Kenya · 🇹🇿 Tanzania · 🇺🇬 Uganda

## Tech Stack
- Vanilla HTML / CSS / JavaScript
- ChatGPT-4 via RapidAPI
- OpenStreetMap Overpass API for hospital data
- Nginx + Docker for deployment

## ⚠️ Disclaimer
ResQ does **not** replace professional medical advice. Always call emergency services first — **112** in East Africa.
